# Karma Cycle — Todo

Track planned features and work items here. Each item syncs to a GitHub issue automatically on commit.

## Format

Use this format for todo items. The sync script parses it on every commit.

```
- [ ] Title of the feature (#123)        ← linked to issue #123
- [ ] New idea without an issue yet       ← issue created automatically on commit
- [x] Completed item (#45)               ← issue #45 closed automatically
```

Rules:
- `- [ ]` = open todo → creates or keeps issue open
- `- [x]` = done → closes the linked issue
- `(#123)` at the end = already linked to an issue
- No `(#N)` = sync script creates a new issue and appends the number
